## BƯỚC 1: TÌM FILE EXPLORER (Danh sách files)

### 🔍 Nhìn màn hình Cursor của bạn:

```
╔════════════════════════════════════════════════════════════╗
║  THANH BÊN TRÁI (màu tối)          |  KHU VỰC CHÍNH      ║
║  ↓                                  |                      ║
║  Ở đây có các ICON                 |  Chỗ này hiện code   ║
║  (biểu tượng hình vuông)           |  hoặc chat với AI    ║
║                                     |                      ║
╚════════════════════════════════════════════════════════════╝
```

### 👀 TÌM THANH BÊN TRÁI:

**Nó ở đâu?**
- Ở **cạnh trái** màn hình
- Có **màu tối** (xám đen hoặc đen)
- **Hẹp** (chỉ rộng khoảng 3-4cm trên màn hình)

**Có những gì?**
- Nhiều **biểu tượng nhỏ** (icon)
- Xếp **dọc từ trên xuống dưới**
- Mỗi icon có **hình khác nhau**

## ❓ CÂU HỎI THƯỜNG GẶP

### Q1: "Tôi không thấy thanh bên trái?"

**A:** Có thể đã bị ẩn:
- Nhấn `Ctrl + B` (Windows) hoặc `Cmd + B` (Mac)
- Hoặc menu View → Show Side Bar

### Q2: "Không thấy file ZIP?"

**A:** 
- Scroll xuống trong danh sách
- Hoặc search: `Ctrl + P` → gõ "zip"
- Hoặc click refresh (icon ↻)

### Q3: "Click chuột phải không có menu Download?"

**A:**
- Thử click chuột trái → xem có nút Download không
- Hoặc drag file ra Desktop
- Hoặc cho tôi biết để tôi hướng dẫn cách khác

### Q4: "File downloaded nhưng không tìm thấy?"

**A:**
- Windows: Nhấn `Win + R` → gõ `downloads` → Enter
- Mac: Finder → Go → Downloads
- Hoặc Chrome: `Ctrl + J` → xem downloads

### Q5: "Giải nén xong không thấy folder?"

**A:**
- Check cùng chỗ với file ZIP
- Hoặc Desktop
- Hoặc search "threads-downloader"
